Biografia

Bacharel em Sistemas de Informação pelo IFES.

Atua como desenvolvedor desde 2013.

Trabalhou com projetos Java Web com Java EE e Framework Jaguar, Spring MVC e Spring Boot. Em desenvolvimento Mobile já desenvolveu Android Nativo e Hibrido, sendo hibrido com React Native e Ionic, conta também com curso de iOS em Objective-c pela Caelum. No Front-end tem projetos desde 2016 com Angular, e é um admirador de Vuejs.
